LabPack LabVIEW Library

November 2017

Thank you for using the LabPack LabVIEW library, version 0.1.0. LabPack is a LabVIEW library for encoding and decoding MessagePack data.

Installation

This section documents installation instructions for LabPack LabVIEW library.

  1. Download the vipm package of the LabPack LabVIEW library.
  2. Open it with VI Package Manager (available for free at www.jki.net/vipm) and follow on-screen instructions.
  3. Restart LabVIEW.

Software Support in LabPack LabVIEW Library

Operating System Support

The following operating systems are supported in the LabPack LabVIEW Library 1.0.0:

  • Windows
  • Mac OS
  • Linux
  • Linux RT

Note, this library uses a shared library to encode and decode MessagePack data. Pre-compiled shared libraries for all listed systems are provided with the vipm.
