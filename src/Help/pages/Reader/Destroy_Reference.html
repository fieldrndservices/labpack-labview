Destroy Reference VI

Owning Palette: Reader VIs

Requires: LabPack

Destroys a reference to a Reader class object. This may free resources associated with the reference but not necessarily the object.

Reader In is a Data Value Reference (DVR) to an object of the Reader class.
Error In describes error conditions that occur before this node runs. With the following exception, this input provides standard error in functionality.

This node runs normally even if an error occurred before this node runs.
Reader Out is an object of the Reader class.
Error Out contains error information. This output provides standard error out functionality.
